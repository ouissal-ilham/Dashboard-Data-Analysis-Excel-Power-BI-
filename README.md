# Sales Data Analysis Dashboard

## 📊 Project Overview
This project analyzes a sales dataset containing information about orders, customers, products, and regions.  
The goal is to explore sales performance and generate business insights using an interactive dashboard built with Power BI.

---

## 📂 Dataset Description
The dataset includes the following key features:

- Order ID, Order Date, Ship Date
- Customer Information (ID, Name, Segment)
- Location (Country, City, State, Region)
- Product Details (Category, Sub-Category, Product Name)
- Sales Values

This dataset allows analysis of sales trends, customer behavior, and regional performance.

---

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Analyze sales performance across regions and categories
- Identify top-performing products and customers
- Build an interactive dashboard for decision-making

---

## 🛠️ Tools Used
- Power BI
- CSV dataset
- Data cleaning and transformation

---

## 📈 Dashboard Features
- Total Sales KPI
- Sales by Region
- Sales over Time (trend analysis)
- Sales by Category and Sub-Category
- Top Products by Sales
- Interactive filters (Region, Date)

---

## 📊 Key Insights
- Identified regions with the highest sales performance
- Observed trends and seasonal variations in sales
- Highlighted best-selling product categories
- Detected differences in customer segments
